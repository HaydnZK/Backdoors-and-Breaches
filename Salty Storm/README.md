# Salty Storm (In Development)
## Operational Overview
This scenario shifts the defensive focus away from standard endpoint analysis and cloud single sign-on logs, moving straight to the enterprise network edge. It simulates a long-term, low-and-slow espionage campaign where the threat actor bypasses traditional Endpoint Detection and Response (EDR) entirely by operating directly out of the organization's core networking appliances and routing infrastructure.

Will the attacker follow the standard playbook and deploy loud, external malware components, or will they take a much quieter approach? They might leverage hidden configurations, protocol abuse, and living-off-the-land techniques straight within the firmware layer to intercept data streams and maintain a silent foothold for years without anyone noticing.
